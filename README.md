# actionfiguremanga.it

Base repository statica pronta per GitHub Pages o deploy tradizionale.

## Struttura
- `/` home con categorie manga
- `/dragonball/` pagina categoria
- `/dragonball/[personaggio]/` money page personaggio
- `/blog/[keyword]/` articoli long tail

## Come usarlo
1. Carica tutto il contenuto su GitHub.
2. Sostituisci i link `#` nei box Amazon con i tuoi link affiliati reali.
3. Personalizza testi, immagini, meta description e contenuti articolo.
4. Aggiungi nuove cartelle seguendo lo stesso schema.
